# quickSwitch 使用教程

quickSwitch 是一个**轻量的 macOS 桌面悬浮快捷条**:把你常用的 App、文件、文件夹、网页"钉"在一条小条上,点一下就切过去/打开。它不占程序坞、不占菜单栏,可以拖到屏幕任意位置,还能贴边自动隐藏、横竖切换。

---

## 一、启动与退出

- **启动**:双击 `quickSwitch.app`(或开发时在项目目录运行 `swift run quickSwitch`)。
- ⚠️ **首次打开被系统拦下?** 如果 `.app` 是下载/AirDrop/微信传来的,macOS 会提示"无法打开"。解决:**系统设置 → 隐私与安全性 → 找到 quickSwitch → 「仍要打开」**;或在终端运行 `xattr -dr com.apple.quarantine quickSwitch.app` 后再打开。
- 它是个**后台挂件**:不会出现在程序坞、也不在 `⌘Tab` 里。屏幕上有那条小条,**菜单栏右上角还有一个网格小图标**(兜底入口:设置 / 教程 / 退出 / 把条移回屏幕中央)。
- **退出**:右键小条空白处 →「退出 quickSwitch」,或点菜单栏图标 → 退出。
- 想开机自动启动:右键 →「设置…」→ 打开「开机自启」。

启动后小条默认在屏幕顶部中间(之后会记住你上次放的位置)。第一次是空的,有一行提示和一个 `+`;首次启动还会自动弹出本教程。

## 二、添加条目(四类)

| 类型 | 怎么加 | 点击后 |
|---|---|---|
| **应用** | 从「访达 → 应用程序」把 App 拖到条上;或 `+`;或右键 →「添加正在运行的应用」 | 切换到该 App(没开则启动) |
| **文件** | 从访达把文件拖到条上;或 `+` 选中 | 用默认程序打开 |
| **文件夹** | 从访达把文件夹拖到条上;或 `+` 选中 | 在访达中打开 |
| **网页** | 右键 →「添加网址…」粘贴链接;或拖浏览器**地址栏的网站图标** | 用默认浏览器打开 |

**最省事的方式**:右键小条空白处,菜单里有
- **添加正在运行的应用** —— 列出你当前开着的 App,点一下就加(程序坞里有的基本都在这);
- **添加文件 / 应用…** —— 弹文件选择器,可多选;
- **添加网址…** —— 粘贴链接(会自动读你剪贴板里的网址预填)。

> ⚠️ 两个 macOS 的限制(不是 bug):
> - **从程序坞直接拖 App 进来无效** —— 系统不把程序坞图标交给第三方窗口。请改用「添加正在运行的应用」或从访达拖。
> - **拖浏览器标签页无效** —— 浏览器会把标签撕成新窗口。请拖**地址栏的网址**,或用「添加网址…」。

## 三、日常操作

| 想做什么 | 操作 |
|---|---|
| **切换 / 打开** | 左键点图标(按下会有缩小反馈;启动中图标会变暗) |
| **回到上一个 App** | 再点一下**已在前台**的那个 App 图标 → 它会隐藏,回落到上一个 App(可在设置里关) |
| **调整顺序** | 按住某个图标拖到另一个上,交换位置 |
| **移动整条** | 按住条上**任意空白处**拖动(图标除外) |
| **重命名** | 右键图标 →「重命名…」(网页名字奇怪时很有用) |
| **移除** | 右键图标 →「移除」 |
| **看名字** | 鼠标悬停图标(光标旁会浮出名字,横竖向都有) |

口诀:**点图标=打开,拖图标=排序,拖空白=移动整条**。

### 全局快捷键(双手不离键盘)

| 快捷键 | 作用 |
|---|---|
| `⌥Space` | **唤出**:条会弹一下并在前 9 个图标下显示数字角标;贴边藏着时滑出(再按滑回)。悬浮时不会隐藏条 |
| `⌥1` ~ `⌥9` | **直接打开**条上第 1~9 个条目,连条都不用唤出 |

组合键在「设置 → 快捷键」里**点击即可重新录制**(按任意带修饰键的组合);两个功能都可独立关闭。实现走系统标准热键接口,**无需辅助功能权限**。

## 四、贴边自动隐藏

把整条**拖到屏幕最左边或最右边**松手,它会滑出去藏起来,只在边缘留一条细痕:
- 鼠标**靠近那条细痕**,它会自动滑出来;
- 鼠标移开,过一会儿又缩回去;
- 拖东西过去时也会自动弹出来,方便你放上去;
- 想拉回中间:让它弹出后,按住空白处把它拖回屏幕中央。

放在屏幕中间时则保持常显,不会乱藏。

## 五、设置窗口(样式与布局)

右键小条空白处 →「**设置…**」,打开设置面板:

**布局**
- **方向**:横向 / 竖向。
- **显示 + 按钮**:关掉后条上就没有 `+` 了(添加改用右键菜单)。

**尺寸与样式**(滑块,实时生效)
- **图标大小**、**圆角**、**背景透明度**、**图标间距**、**内边距**。
- 「恢复默认样式」可一键还原。

**行为**
- **窗口置顶**:是否始终浮在其他窗口之上。
- **菜单栏图标**:右上角的兜底入口(设置/教程/退出/移回中央),不想要可关。
- **点击前台应用时隐藏它**:再点已在前台的 App 会隐藏它、回到上一个 App(A↔B 往返)。
- **开机自启**:登录时自动启动。

所有设置都会自动保存,下次启动沿用。

## 六、排查:某个 App 点了切不过去?

quickSwitch 切换用的是和程序坞一样的系统方式,但极少数 App 仍可能因所在「桌面/Space」或窗口状态切换异常。排查方法:

1. 打开「终端」,运行:
   ```bash
   log stream --predicate 'subsystem == "com.shiqi.quickSwitch"' --level info
   ```
2. 在 quickSwitch 上点那个**失败的图标**,看终端打印:
   - `openApp <id> -> urlForApplication is NIL` → 系统不认识这个 App(建议移除后用「添加正在运行的应用」重新加);
   - `openApp <id> FAILED: …` → 后面是具体错误;
   - `openApp <id> -> ok` 但没切过去 → 多半是该 App 在别的 Space / 全屏,属系统层面问题。
3. 看完按 `Ctrl+C` 停止。

## 七、关于隐私

quickSwitch 只用系统标准接口打开/切换 App(`NSWorkspace`),**不需要辅助功能、录屏、自动化等敏感权限**,也不收集任何数据。
